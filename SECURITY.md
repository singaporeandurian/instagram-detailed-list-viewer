# Security Policy

## Supported Versions

Check [GitHub Releases](https://github.com/singaporeandurian/instagram-detailed-list-viewer/releases) for supported public builds. If no public build is listed yet, this repository should be treated as a documentation and support repository.

## Reporting A Vulnerability

Do not post sensitive security issues publicly.

Report security concerns privately through the preferred contact channel listed on the official product page or GitHub profile. Include:

- App version
- Operating system
- Clear reproduction steps
- Whether the issue involves local files, imported exports, AI-powered insights, or network requests
- Any relevant logs with private Instagram data removed

## Sensitive Data Reminder

Instagram exports can contain personal data. Do not attach full exports to public issues. Redact usernames, messages, emails, phone numbers, links, and any other private information before sharing screenshots or logs.

## Scope

Security reports should focus on:

- Unsafe handling of local export files
- Unexpected network transmission of private data
- Credential prompts that should not exist
- Insecure update or download behavior
- Crashes that expose sensitive information

Instagram Detailed List Viewer is independent software and is not affiliated with Instagram or Meta.
