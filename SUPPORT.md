# Support

Use this repository for public documentation, issue tracking, and release notes for Instagram Detailed List Viewer.

## Before Opening An Issue

Check:

- [Getting started](docs/getting-started.md)
- [File formats](docs/file-formats.md)
- [Troubleshooting](docs/troubleshooting.md)
- [FAQ](docs/faq.md)

## Bug Reports

Open a bug report when:

- An Instagram export file does not load.
- A followers or following list appears empty.
- A comparison report looks incorrect.
- The app crashes or freezes.
- AI-powered insights do not match the selected list context.

Include:

- App version
- Operating system
- Export format, such as JSON or HTML
- Whether the export was zipped or extracted
- Which list failed, such as followers, following, or pending requests
- A redacted screenshot if helpful

Do not attach private Instagram exports publicly.

## Feature Requests

Open a feature request when you want support for:

- A new Instagram export file type
- A new comparison view
- Better AI-powered insights
- Additional privacy controls
- Exporting reports to CSV, JSON, or PDF

## Private Or Sensitive Issues

For security issues, do not open a public issue. See [SECURITY.md](SECURITY.md).
